# Quant 101 — Introduction à la Finance Quantitative en Python

Bienvenue dans **Quant 101**, une série d’ateliers pratiques conçus pour te donner une base solide en **finance quantitative**, en liant théorie financière et implémentation concrète en Python.

Chaque atelier introduit un concept clé — des rendements aux stratégies de trading avancées — à travers des notebooks interactifs, accessibles publiquement et faciles à exécuter via **Google Colab**.

---

## Objectif de la série
- Comprendre les **fondements analytiques** de la finance de marché.  
- Apprendre à **manipuler, mesurer et modéliser** les données financières.  
- Construire des **outils concrets** : visualisations, indicateurs, tableaux de bord, et backtests.  
- Développer une **rigueur quantitative** réutilisable dans tes études, projets ou futurs emplois.

---

## Structure des ateliers

Chaque atelier est structuré autour de :
1. **Introduction théorique** : Concepts financiers et mathématiques clés.  
2. **Implémentation en Python** : Étapes guidées pour coder les analyses et visualisations.  
3. **Exercices pratiques** : Applications concrètes pour renforcer l’apprentissage.  
4. **Démo de synthèse** : Mini-démo pour résumer les compétences acquises.


---

## Comment utiliser les notebooks
1. Ouvre le notebook de l’atelier souhaité.  
2. Clique sur le badge **“Open in Colab”** pour l’exécuter directement dans ton navigateur.  
3. Aucun environnement local n’est requis : tout fonctionne sur Google Colab avec un compte Google.  

Chaque dossier contient :
- Un **notebook `.ipynb` principal** avec tout le code et les explications.  
- Un **README dédié** présentant les objectifs, le contenu et les livrables.  
- Un **dossier `/data` ** (s'il y a des fichiers nécessaires).

---

##  Outils utilisés
- **Python** : Pandas, NumPy, Matplotlib, Seaborn  
- **Sources de données** : yfinance, fichiers CSV externes  
- **Jupyter / Google Colab** pour l’interactivité  
- **GitHub** pour la gestion et le partage des projets
